Responsive User Interfaces With Bootstrap And ReactJS

Learn How To Combine Bootstrap With ReactJS To Create Responsive User Interfaces

Posted by Heikki Kupiainen/Oppikone on 2017-01-04

I wrote earlier a blog post about creating a simple app with ReactJS. ReactJS is a template engine that pretends to be something greater than just another template engine like it is. But just like with any template engine, at the end of the day you'll have some generated html code and a heavy burden on your shoulders because you must also make it look fashionable. And you are still just an engineer, nothing greater. Therefore I tell you, do not worry about your layout. Because Bootstrap can take away this burden from your shoulders!

But before we start, please note that this is not an article about a certain "React-Bootstrap library" that someone has written. There is absolutely no need to use any additional library or hock-us-pock-us to incorporate Bootstrap with ReactJS. There is a strong wind of over-engineering and wheel-reinvention in ReactJS community. Its purpose is just to make simple things appear complicated - just as ReactJS itself is just a new fashion of a template engine with a bunch of new buzzwords for age old concepts. But let's not worry about it for now.
